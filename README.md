# Enterprise ERP Project (Multi-Tenant)

## Tech Stack
- **Backend**: Laravel (PHP)
- **Database**: PostgreSQL
- **Frontend**: NuxtJS 3 (Vue 3, PrimeVue)

## Project Structure
- `/backend`: Laravel Multi-tenant API.
- `/frontend`: NuxtJS Client application.

## Key Architecture
- **Tenancy**: Multi-database isolation using `stancl/tenancy`.
- **UI System**: PrimeVue with customized design tokens.
- **Auth**: Laravel Passport with multi-tenant scoping.
